## 🌟 WELCOME
This guide is for parents, teachers, and content creators who want to add new learning content to the app **without writing complex code**.

---

## 📂 HOW TO ADD NEW WORDS
Project data lives in "Data Modules" inside `frontend/src/js/data/`.

### Step 1: Find the right file
- To add animals, open `mera_sansar/animals_wild.js`.
- To add a new Fruit, open `khana_peena/fruits.js`.

### Step 2: Add a new entry
Copy an existing line and change the information inside the `{ }` brackets.
```javascript
{
  word: "आराम",     // Type the Hindi word here
  english: "Rest",   // Type the English identifier here
  icon: "🛌",       // Paste an Emoji or Icon here
  audio: "rest"      // Type the name of the sound file (without .mp3)
}
```

### Step 3: Add the Audio
Save your recorded MP3 file into the corresponding folder (e.g., `public/assets/audio/mera_sansar/`). Ensure the name matches the `audio` key you typed in Step 2.

---

## 🎨 PICKING ICONS
- **Emojis**: Built-in phone/computer emojis are best for simple recognition.
- **Custom Images**: If an emoji is unavailable, request a developer to add a custom `.png` to the `assets/images/` folder.

---

## ✅ FINAL CHECK
1. Is the word correctly spelled in Hindi?
2. Does the audio file play clearly?
3. Does the icon clearly represent the word?
