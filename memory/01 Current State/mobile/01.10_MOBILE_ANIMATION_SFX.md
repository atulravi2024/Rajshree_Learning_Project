# ✨ MOBILE ANIMATIONS & SFX (v1.0)

## 1. High-Density Confetti Engine
Triggered by `triggerCelebration()` upon completing a category.

### A. Piece Logic
- **Shapes**: `square`, `circle`, `triangle`, `star`.
- **Colors**: A vibrant 8-color randomized palette.
- **Physics**: Random drift (`-150px` to `150px`) and duration (`2.5s` to `5.0s`).
- **Density**: Spawns 120 initial pieces + 40 pieces/second via `setInterval`.

## 2. Interaction Soundscapes
### A. Wind Feedback (`storm-wind.mp3`)
- **Pitch/Speed Control**: Audio playback speed is synchronized with `state.playbackSpeed`.
- **Dynamic Volume**: Calculated via `getEffectiveVolume('sfx')` (Master * SFX_Multiplier).

### B. Celebration Sfx (`reward_excellent.mp3`)
- Triggered alongside the confetti engine for a multi-sensory reward experience.

## 3. UI Motion Standards
- **Nav Transition**: `transform: translateY(100%)` for autohide bar.
- **Slide Animations**: `slideInRight` and `slideInLeft` use `transform: translateX` + `scale(0.95)` to `scale(1.0)` for a spring-like entry effect.
- **Pop Effect**: `.sfx-pop` uses keyframes to scale the container to `1.1x` for instant tactile response.
