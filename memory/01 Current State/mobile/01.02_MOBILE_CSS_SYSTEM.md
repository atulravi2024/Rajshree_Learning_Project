# ЁЯОи MOBILE CSS SYSTEM (v1.0)

## 1. Registry: `frontend/mobile/css/`
The styling system uses CSS Variables (`:root`) and modular files for high maintainability.

### A. Core Tokens (`main.css`)
- **Variables**: `--primary-color`, `--glass-bg`, `--surface-color`.
- **Z-Index Map**:
  - `mobile-nav`: 1000
  - `mobile-overlay`: 2000 (Welcome), 3000 (Menu)
  - `confetti-container`: 9999
- **Animations**:
  - `.slide-in-right`, `.slide-in-left`: Horizontal nav animations.
  - `.slide-in-top`, `.slide-in-bottom`: Vertical nav animations.
  - `.sfx-pop`: Interaction feedback bounce (0.3s).

### B. Settings Infrastructure (`settings_core.css`)
- **Layout**: Uses Flexbox for tab navigation and CSS Grid for safety lock cards.
- **Glassmorphism**: Premium blur effects (`backdrop-filter: blur(15px)`).
- **Draggable Tabs**: Styling support for horizontal scroll regions without visible scrollbars.

---

## 2. Layout Modes & Ratios
### A. Flashcard Container
- **Ratio**: `aspect-ratio: 4/5` (Optimized for 9:16 - 9:21 screen ratios).
- **Border**: 8px solid white surface + 3px outline offset.
- **Flip**: 3D rotateY transformation via `perspective: 1500px`.

### B. Three-in-One View
- **Card Scaling**: Width set to 31%.
- **Typography**: Card letters scaled down to `3.2rem`.
- **Layout**: Supports both horizontal and vertical container flex directions.

---

## 3. Dark Mode Support
- Triggered by `.dark-mode` class on `:root`.
- **Color Overrides**:
  - `--glass-bg`: `rgba(26, 26, 46, 0.9)`
  - `--surface-color`: `#121212`
  - `--text-main`: `#f8fafc`
- Performance: Uses `transition: background 0.5s` for smooth mode switching.
