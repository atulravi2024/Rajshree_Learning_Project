# 📱 MOBILE UI/UX PRINCIPLES (v1.0)

## 1. Single-Handed Design
- All critical navigation (`Next`, `Prev`, `Home`) is grouped in a persistent bottom navbar to ensure accessibility by thumb.
- The **Start Orb** in the Welcome Screen is centered and large (150px) for easy tap without looking.

## 2. Interaction Feedbacks
- **Auditory SFX**: Every button click or card flip triggers a high-fidelity "Wind/Pop" sound effect via `playInteractionSFX()`.
- **Visual Scaling**: Buttons and cards scale by `1.05x` on `:active` touch states.
- **Swipe Gestures**:
  - Horizontal swipe (Left/Right) for Standard layout.
  - Vertical swipe (Up/Down) for Vertical layout modes.

## 3. Visual Hierarchy
- **Typography Focus**: Large Hindi letters (`card-letter`) dominate the view for child readability.
- **Glassmorphism**: Backdrop blur (15px) ensures overlays (Menu, Settings) feel spatially distinct from the content below.
- **Theming**: Dynamic `--glow-color` mirrors the selected theme (e.g., Pink theme → Pink glow).

## 4. Continuity & Persistence
- **State Save**: Every card advance or category change is instantly synced to `localStorage`.
- **Smooth Transitions**: Using `cubic-bezier(0.175, 0.885, 0.32, 1.275)` for "bouncy" premium feeling on all mobile animations.
