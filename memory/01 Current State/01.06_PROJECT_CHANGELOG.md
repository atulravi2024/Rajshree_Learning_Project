## 🏗️ ARCHITECTURE DECISIONS (AUDIT TRAIL)

### Why Memory Was Relocated to Domain 00 (v16.7)
- Promoting `00.00_MEMORY_MAP.js` to the System range established an absolute governance root.
- Prevents technical debt where a secondary domain (Current State) was hosting the primary registry.

### Why CSS Was Modularized (v13.0)
- Single 19KB `style.css` was causing layout regressions when adding theme features.
- Modularized into 10 specialized files to enable clean theme switching in Phase 2.

### Why Context Window Fragmentation (v14.0)
- Large documentation files (>20KB) were exceeding AI context window limits during audits.
- Fragmented into on-demand details to ensure 100% detail coverage with minimal token usage.
