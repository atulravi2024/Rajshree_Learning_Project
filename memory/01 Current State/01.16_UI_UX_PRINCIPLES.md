## 🌟 CORE PHILOSOPHY

### 1. Gentle Learning
The application should never feel overwhelming.
- **Audio Fade**: All audio starts with a 300ms fade-in to avoid startling the child.
- **Soft Palette**: Use HSL-tailored pastel colors to reduce eye strain.
- **Clear Feedback**: Every interaction must have a clear visual or auditory response.

### 2. Safety First (Child-Proofing)
The UI is a "Sanctuary" where the child cannot unintentionally "break" the experience or exit to system settings.
- **Zero Accidental Exits**: All destructive or configuration actions are behind the Parental Gate.
- **No Scrolling**: The "Fit-Screen" philosophy ensures all content is visible at once, preventing navigational confusion.

### 3. High Engagement (Magical UI)
The app should feel "Alive" and interactive to maintain a child's short attention span.
- **Micro-Animations**: Emojis wiggle on hover, the logo "dances," and stars celebrate high numbers.
- **Premium Depth**: Use Glassmorphism and 3D transforms to create a modern, high-quality feel that distinguishes it from generic educational tools.

## 🛠️ INTERACTION STANDARDS

| Feature | Standard | Purpose |
|:---|:---|:---|
| **Hover Scale** | `1.05x` with 0.3s transition | Subtle feedback for target selection. |
| **Audio Lock** | 100% UI Lock during playback | Ensures the child hears the full label before moving on. |
| **Auto-Slideshow**| 3.5s delay + audio duration | Balances automated learning with information retention. |
