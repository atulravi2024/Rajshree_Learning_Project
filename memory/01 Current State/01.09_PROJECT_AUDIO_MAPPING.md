## SECTION C — AUDIO PLAYBACK LOGIC (`helpers.js`)

### Path Construction
```javascript
const finalAudioPath = file.startsWith('assets') ? file : window.AUDIO_BASE_PATH + file;
// window.AUDIO_BASE_PATH = 'assets/audio/'
```

### Visual Feedback during Playback
- **Class `playing`**: Applied to card while audio is active.
- **Progress Bar**: `.progress-bar` width is updated via `timeupdate` event.
- **Slideshow**: `ended` event triggers `handleSlideshowTransition()` if slideshow is active.
