## 📐 CORE ARCHITECTURAL RULES

### 1. Flash Card Layout Logic
- **Absolute Anchoring**: The **Letter** and **Word** (Text Group) must ALWAYS be stuck to the absolute bottom of the flashcard padding zone.
- **Icon Positioning**: The upper/center section of the card is reserved for icons. Use `margin-top: auto` on the text group to ensure it stays at the bottom regardless of icon count or size.
- **Layout Consistency**: This bottom-fixed rule applies to ALL views: **1x1 (Single)**, **3-Card (Grid)**, **All (Scroll)**, and **Fit Screen**.

### 2. Dynamic Scaling & Layout Modes
| Layout Mode | Icon Size (Emoji) | Letter Size | Word Size | Columns | Scroll Direction | Vertical Alignment | Card Height (Fixed) |
|:---|:---:|:---:|:---:|:---:|:---:|:---:|:---:|
| **Single Mode (1x1)** | 22vh | 15vh | 7vh | 1 | N/A | Center | N/A |
| **3-Card Mode** | 15vh | 10vh | 5vh | 3 | N/A | **Strict Center** | **450px** |
| **Scroll Mode (All)** | 10vh | 6vh | 3vh | 4 | **Strictly Vertical** | Top-Start | **250px** |
| **Fit Screen Mode** | 8vh | 5vh | 2.5vh | **Dynamic** | N/A | Stretch/Center | Dynamic |

#### Fit Screen Column Logic:
- **Numbers 1-100**: Locked to **10 columns**.
- **Others**: Dynamic based on count: 9 columns (>36), 8 columns (>25), 6 columns (>16), 5 columns (>9), 4 columns (>4).

### 3. Category & Layout Exclusive Rules
#### A. Global Fit Screen Rules (Most Categories)
- **Front Side**: Only **Icon** and **Single Letter** are visible.
- **Word Visibility**: All words are **HIDDEN** on the front side (`display: none`).
- **Vyanjan (Consonants)**: Icons and Letters scaled down by **30%** (0.7x factor).

#### B. Counting Category (1-100) Specifications
- **Fit Screen Exclusive**: Front side is **ICON-ONLY**. Hide all text, letters, and numerals.
- **Icon Scaling**: Exactly **50%** of base size.
- **Zero Icon Override**: Zero (**०**) → **Egg icon (🥚)**. (100 remains **💯**).
- **5-Column Grid**: `calc(20% - 12px)` width logic.
- **Phonetic Data**: 100-item `hindiNumbersNames` array for audio-text sync.

#### C. Counting Category (1-10) Specifications
- **Word Visibility**: Words remain **VISIBLE** on front side in Fit Screen mode.

### 4. Application Logic & Stability
- **Intelligent Slideshow Protocol**:
    - **Total UI Lockdown**: Category Icons, Home, Layout, Side Arrows → **GREYED OUT** (Opacity 0.25-0.4) and **DISABLED**.
    - **Force-Next Logic**: `next(true)` internal trigger bypasses manual navigation locks.
    - **Double-Delay Timing**: 1s before flip → 1s after audio → 1s after page switch.
    - **Page Reset**: Container resets `scrollTop/Left = 0` on every page change.
    - **Completion Reward**: Plays `system/effects/reward_excellent.mp3` with alert "Category Complete! Great job!"
- **UI Interaction Rules**:
    - **Dropdowns**: 1000ms closing timeout.
    - **Volume Icon**: Dynamic (🔇, 🔉, 🔊) based on values (0, <0.5, >=0.5).
    - **Progress Bar**: Linear tracking during audio playback.
- **Animations**: Idle = `playful-bounce` (3s). Hover = `emoji-wiggle` (0.3s, overrides bounce).
- **Stability**: Grid uses `align-content: center` + `height: 100%` + `grid-template-rows` reset.
