## 🏗️ ARCHITECTURAL TERMS

### Step [1-5] Overlay
The multi-layered category selection system used by the router.

### Learning Zone
The final interface (Step 5) where active flashcards are displayed.

### Neural Assets
Media files (Audio/Icons) linked to data objects rather than hardcoded in HTML.

### Frontier Protocol (v2.0)
The high-fidelity internal audit engine used for system oversight and health monitoring.

### Modular Settings
The premium settings interface loaded via an isolated iframe to ensure role-based configuration.

---

## 🛡️ SAFETY & SECURITY TERMS

### Parental Gate
A 3-second continuous hold requirement on specific UI elements to prevent accidental child interaction.

### Spam Guard
A 400ms time-lock on interaction buttons to prevent rapid clicking and audio overlapping.

### PostMessage Sync
The secure cross-window communication method used to synchronize settings between the iframe and the main app.

---

## 🎨 UI & INTERACTION TERMS

### Glassmorphism
The semi-transparent visual style using `backdrop-filter: blur(20px)` for a premium feel.

### 3D Flip System
The animation system allowing flashcards to rotate 180 degrees to reveal their back side.

### Fit-Screen Grid
A dynamic layout mode where card sizes are calculated to fit the screen without scrolling.
