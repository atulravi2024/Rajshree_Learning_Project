## рЯУЬ SCRIPT LOADING ARCHITECTURE (v17.2)

The application loads scripts from the `frontend/src/js/` directory.

### 1. Data Modules (Atomic Content)
- **Varnamala**: `swar.js`, `vyanjan.js`, `samyukt.js`, `matra.js`
- **Maths (Ganit)**: `numbers_10.js` to `tables_10_m2.js`, `shapes.js`, `comparisons.js`
- **Mera Sansar**: 25+ modules including `family.js`, `animals_wild.js`, `fruits.js`, `space.js`, etc.
- **Colors (Rangon ka Sansar)**: 8 nested modules.
- **Time/Nature (Samay/Prakriti)**: 4 modules.

### 2. Core Logic (Execution Order)
1. `core/main.js`: Global state & configuration.
2. `utils/child-safety.js`: Parental gate and safety protocols.
3. `core/init.js`: Bootloader and event listeners.
4. `utils/helpers.js`: Shared utility functions.
5. `ui/display.js`: Card rendering and layout management.
6. `navigation/router.js`: Navigation flow and category switching.
7. `ui/animations.js`: UI effects and transitions.
8. `ui/settings-modal.js`: Iframe-based settings modal handler.
9. `slideshow/controller.js`: Auto-play and timer logic.
