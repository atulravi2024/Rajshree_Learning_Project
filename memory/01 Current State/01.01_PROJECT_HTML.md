- **Floating Icons**: `#bg-icons` contains spans dynamically generated by JS.

- **Step 2 (Title Category Screen)** `#step-2`:

    - 🕐 समय और रंग (`.magic`) → `showMainCategory('samay')`
- **Step 3 (Main Category Screen)** `#step-3`:
  - Dynamic container `#step3-container` + title `#step3-title`.
  - Back button → `goBackTo(2)`.
- **Step 4 (Sub Category Screen)** `#step-4`:
  - Dynamic container `#step4-container` + title `#step4-title`.
  - Back button → `goBackTo(3)`.
- **Step 5 (Deep Category Screen)** `#step-5`:
  - Dynamic container `#step5-container` + title `#step5-title`.

### 4. Main Learning Zone (`.main-content`)
- Wraps `#learning-zone` (`.learning-wrapper`, `.hidden`).
- Custom Navigation Arrows (CSS-drawn, not SVG in HTML — SVGs injected via JS in `display.js`):
