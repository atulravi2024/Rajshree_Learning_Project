## HTML — Unified Modular Architecture
- **Entry Point**: `frontend/public/index.html`

### 1. Head Section
- **Language**: `hi` (Hindi)
- **Meta Tags**: `charset="UTF-8"`, `viewport="width=device-width, initial-scale=1.0"`
- **Title**: `राजश्री हिंदी लर्निंग | Rajshree Hindi Learning 🎒✨`
- **Stylesheets**: Links to `../src/css/main.css`

### 2. Compact Navbar (`#main-nav`, `.navbar`, `.hidden`)
- **Logo Area**:
  - Contains `.baby-icon` (👶) and `.logo-text` (Hindi and English titles).
- **Navigation Menu Container**:
  - Horizontal list (`.nav-menu`) of main categories.
  - **Category Links**: 
    - `#nav-varnamala` (📖)
    - `#nav-sankhya` (🧮)
    - `#nav-names` (🌍)
    - `#nav-samay` (☀️)
  - **Action Links**:
    - `#nav-home` (🏠) triggers `goHome()`
    - `#nav-slideshow` (🎬) triggers `toggleSlideshow()`
  - **Dropdowns**: Layout Control (📐) and Volume (🔊).

### 3. Overlay System (`#overlay`)
- **Floating Icons**: `#bg-icons` contains spans dynamically generated by JS.
- **Step 1 (Welcome Screen)**:
  - Welcome Title, greeting text, and `.start-orb` button (🚀).
- **Step 2 (Title Category Screen)**:
  - `.choice-grid` with buttons for 4 main categories.
  - Managed by `showMainCategory()` function.
- **Step 3 (Main Category Screen)**:
  - Dynamic container `#step3-container`.
  - Back button routing to Step 2.
- **Step 4 (Sub Category Screen)**:
  - Dynamic container `#step4-container`.
  - Back button routing to Step 3.
- **Step 5 (Deep Category Screen)**:
  - Added for nested levels (like `colors_world_main`).
  - Dynamic container `#step5-container`.
  - Back button routing to Step 4.

### 4. Main Learning Zone (`.main-content`)
- Wraps `#learning-zone` (`.learning-wrapper`, `.hidden`).
- Custom Navigation Arrows:
  - `#prev-btn` (Left Arrow)
  - `#next-btn` (Right Arrow)
- `#master-container` (Card grid area). The DOM contents inside this container are fully dynamic driven by JS.
