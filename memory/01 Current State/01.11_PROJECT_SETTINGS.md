## 🛡️ PARENTAL GATE
Accessing the Settings menu via the ⚙️ icon in the Navbar is protected by a **3-second continuous hold**.
- **Lock Type**: Time-based interaction hold.
- **Visual Feedback**: Progress ring/overlay managed by `window.ChildSafetyLock`.
- **Purpose**: Prevents accidental changes to volume or safety mode.
