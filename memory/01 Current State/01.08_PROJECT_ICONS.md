## 7. Flashcard Icons (Special Types in Data Files)

### Khali Icon (Empty/Placeholder)
- **Used for**: Vyanjan characters with no common word association
- **Characters**: ङ, ञ, ण
- **Data property**: `type: 'red-circle'`, `emoji: ''`, `word: 'खाली'`
- **Rendered as**: Double circle (outer red `#F44336` with border, inner dashed `#333` circle) — CSS generated in `display.js`
- **Image asset**: `frontend/public/assets/images/icons/khali_icon.png` (76KB) — exists but not currently used (CSS rendering used instead)

### Special Card Types (Back Side, from `display.js`)
| Data Property | Back-Side Rendering |
|:---|:---|
| `data.image` | `<img>` with `.flash-image` class |
| `data.content` | Text div (6vh font) |
| `data.value` | Number div (15vh font) |
| `data.color` | Full-width color swatch |
| `data.type === 'circle'` | Circle shape with color fill |
| `data.type === 'square'` | Square shape with color fill |
| `data.type === 'triangle'` | CSS triangle with color fill |
| `data.type === 'star'` | ⭐ emoji with color |
| `data.type === 'big-small'` | Two boxes (comparison) |
| `data.type === 'tall-short'` | Two bars (comparison) |
| *default* | Letter in large purple text |
