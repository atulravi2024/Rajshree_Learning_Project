## 🚀 PERFORMANCE TARGETS
To maintain a "Premium & Fluid" feel, the application adheres to these targets:

- **LCP (Largest Contentful Paint)**: `< 1.2s` (Initial categories visible).
- **FID (First Input Delay)**: `< 100ms` (Navbar interactions).
- **CLS (Cumulative Layout Shift)**: `0` (Fixed UI grid prevents shifting).
- **Interaction Response**: < 50ms for card hover/reveal.

---

## 🛠️ OPTIMIZATION RULES
1. **Audio**: No individual `.mp3` file should exceed `1.2 MB`. Target 128kbps for clear voice.
2. **Icons**: Unicode Emojis are preferred over Images/SVGs to minimize HTTP requests.
3. **Scripts**: Use deferred loading or purely functional modules to avoid blocking the main thread.
4. **Scrolling**: Zero-scroll philosophy (Fit-Screen) minimizes repaints and composite costs.
