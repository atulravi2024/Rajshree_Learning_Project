## 📲 Phase 4: Full PWA Conversion (Web-to-App)
**Goal:** Deploy a `manifest.json` and `service-worker.js` to enable "Add to Home Screen" and offline capabilities.

### 📋 Deep Technical Blueprint

#### 1. Manifest Schema (`frontend/public/manifest.json`) — [NEW FILE]
```json
{
  "name": "Rajshree Learning",
  "short_name": "Rajshree",
  "description": "Interactive Hindi Flashcards for Kids",
  "start_url": "/index.html",
  "display": "standalone",
  "background_color": "#FFDEE9",
  "theme_color": "#FF1493",
  "orientation": "landscape",
  "icons": [
    { "src": "assets/images/icons/icon-192.png", "sizes": "192x192", "type": "image/png" },
    { "src": "assets/images/icons/icon-512.png", "sizes": "512x512", "type": "image/png" }
  ]
}
```

#### 2. Offline Caching Strategy (`frontend/public/sw.js`) — [NEW FILE]
```javascript
const CACHE_NAME = 'rajshree-v1';
const ASSETS = [
  '/', '/index.html', '/css/main.css', '/js/core/main.js',
  '/assets/audio/system/welcome_short.mp3',
  '/assets/audio/system/effects/reward_excellent.mp3'
];

self.addEventListener('install', (e) => {
  e.waitUntil(caches.open(CACHE_NAME).then(c => c.addAll(ASSETS)));
});

self.addEventListener('fetch', (e) => {
  e.respondWith(caches.match(e.request).then(res => res || fetch(e.request)));
});
```

#### 3. Registration Logic (`frontend/src/js/core/init.js`)
Add to the end of the file:
```javascript
if ('serviceWorker' in navigator) {
  window.addEventListener('load', () => {
    navigator.serviceWorker.register('/sw.js')
      .then(reg => console.log('SW Registered'))
      .catch(err => console.log('SW Registration Failed'));
  });
}
```

### 📋 Deep Technical Blueprint

#### 1. Data Modification (Example: `mera_sansar/festivals.js`)
Add a `isPremium` boolean to data modules:
```javascript
window.RAJSHREE_DATA = window.RAJSHREE_DATA || {};
window.RAJSHREE_DATA.festivals_is_premium = true; // Use a convention
```

#### 2. Navigation Watcher (`navigation/router.js`)
Update `showSubCategory` or `finishSetup` to check for premium status:
```javascript
function checkPremiumAccess(category) {
    if (window.RAJSHREE_DATA[category + '_is_premium'] && !window.userHasPremium) {
        showPaywallOverlay();
        return false;
    }
    return true;
}

// Inside finishSetup(category):
if (!checkPremiumAccess(category)) return;
```
