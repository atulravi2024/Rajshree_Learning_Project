## 🖲️ SENTINEL TOOLTIP ENGINE (`initSentinelTooltips`)

The dashboard features a professional-grade floating tooltip system designed for sub-pixel accuracy and high-fidelity project oversight.

### 1. The Floating Proxy (`#sentinel-tooltip-float`)
- **Mechanism**: A single document-level DOM element that follows the cursor with a 300ms activation delay for "Premium feel."
- **Delegation**: Uses global mouseover/mouseout listeners for `[data-tooltip], .card-icon, .close-console, .expand-btn`.

### 2. Collision & Flip Math
- **Placement**: Default: `rect.bottom + 8`.
- **Vertical Flip**: Threshold check: `if (top + tooltipRect.height > window.innerHeight)` -> flip to `rect.top - 8`.
- **Horizontal Clamping**: Central alignment logic with 10px screen-edge padding to prevent bleed-out.
- **Auto-Sync**: Listeners on `scroll` and `resize` immediately hide the tooltip to prevent visual artifacts.
