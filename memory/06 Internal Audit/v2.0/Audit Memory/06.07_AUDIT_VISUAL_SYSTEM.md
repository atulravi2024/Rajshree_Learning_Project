## 🌊 DYNAMIC WAVEFORMS
The "Neural" feel is achieved through randomized CSS variables injected via JS:
- **Element**: `.card-waveform .wave-bar`
- **Logic**: 
  - `animation-duration`: `(1.2 + Math.random() * 0.6)s` (Prevents synchronous "robotic" pulsing).
  - `animation-delay`: `(Math.random() * -2)s` (Ensures bars start at different positions).
- **CSS**: `linear-gradient(to top, var(--cyan-glow), var(--cyan))` with `0.4` opacity.

---

## 📈 ADVANCED CHARTING
Custom extensions to **Chart.js v4.4.1** for a HUD aesthetic:
- **Gfx Pass**: 
  - `shadowBlur: 10`: Adds a neon glow to the data line.
  - `shadowColor: rgba(0, 240, 255, 0.6)`.
- **Metrics**: 
  - `pointRadius: 3` (Base) -> `5` (Hover).
  - `tension: 0.4`: Ensures smooth "organic" spline interpolation.
- **Scale**: `display: false` for Y-axis to maintain a minimalist dashboard profile.

## 📺 CRT & SCANLINE EFFECTS
The "Digital Void" background utilizes a multi-layer CSS approach:
- **Scanlines**: `background: linear-gradient(rgba(18, 16, 16, 0) 50%, rgba(0, 0, 0, 0.25) 50%), linear-gradient(90deg, rgba(255, 0, 0, 0.06), rgba(0, 255, 0, 0.02), rgba(0, 0, 255, 0.06))`
- **Size**: Scaled to `100% 4px` for the vertical CRT line effect.
- **Gradient**: Radial "Void" center with `rgba(1, 10, 18, 1)` edge falloff.
