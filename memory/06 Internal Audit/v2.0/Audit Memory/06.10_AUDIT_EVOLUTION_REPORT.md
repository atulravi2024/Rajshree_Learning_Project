## 🎨 VISUAL & KINETIC UPGRADES

### 1. The "Sentinels"
- **v1.0**: Used simple CSS `pulse` animations for connectivity state.
- **v2.0**: Implemented status-aware animations (`sentinel-aura`, `glitch`, `spin`, `alarm`) that reflect real-time system health.

### 2. CRT Simulation
- **v1.0**: Standard semi-transparent backgrounds.
- **v2.0**: Global `background-position` scanlines at `4px` density, creating a hardware-terminal aesthetic.

### 3. Typography
- **v1.0**: Mixed `Hind` and `Outfit`.
- **v2.0**: Standardized on `Outfit` (Headers) and `Fira Code` (Technical Values), ensuring a uniform HUD profile.
