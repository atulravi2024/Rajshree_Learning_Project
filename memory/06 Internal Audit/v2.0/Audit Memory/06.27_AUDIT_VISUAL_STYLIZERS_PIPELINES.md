## 🗺️ GLOBAL ASSET PIPELINE (GeoJSON)

The Map logic fetches and renders high-density geographic boundaries as stylized 3D vectors.

### 1. Data Sources
- **Countries**: `johan/world.geo.json` (Administrative-0).
- **States/Provinces**: `natural-earth-vector` (Administrative-1).
- **Populated Places**: `natural-earth-vector` (110m resolution).

### 2. Centroid Logic (`getCentroid`)
- **Mechanism**: Iterates through all vertices of a Polygon/MultiPolygon and finds the bounding-box center.
- **Purpose**: Precise placement of Sprite-based Map Labels even for complex, multi-island territories.

## 🛣️ STYLIZED ROAD NETWORK

The system auto-generates a "Data Road" infrastructure to enhance complexity in Section-G and Atlas modes.

- **Mechanism**: Connects each city in `CITY_LABELS` to its next two nearest neighbors.
- **Constraints**: Limited to connections < 4,000 KM to maintain a "Local Transit" look.
- **Renderer**: Uses `THREE.QuadraticBezierCurve3` for a parabolic arc centered at the globe's radius + 0.5 units.
- **Blending**: `THREE.AdditiveBlending` with `0.35` opacity.

## 🎭 VISUAL STYLE REGISTRY

| Style Category | Parameter | Key Registry |
|:---|:---|:---|
| **Path Style** | `SELECTED_LINE_STYLE` | `pulsed`, `solid`, `dotted`, `hyper`, `ghost`, `spike`, `field` |
| **Path Color** | `SELECTED_PATH_COLOR` | `0x00f0ff` (Cyan), `0xff3e3e` (Red), `0x22c55e` (Green), `0x3b82f6` (Blue), `0xfacc15` (Yellow), `0xa78bfa` (Purple), `0xf97316` (Orange), `0xec4899` (Pink) |
| **Route Type** | `SELECTED_PATH_TYPE` | `curve` (Arched), `straight` (Direct), `circle` (Orbital) |

### UI Sync Logic
- **Dropdown State**: Each selector using a `.pointer-menu` toggles the `active` class on click.
- **Palette Logic**: `btn-palette-selector` dynamically updates its `color`, `borderColor`, and `boxShadow` (0 0 10px hex) to match the `SELECTED_PATH_COLOR`.
- **Atomic Refresh**: Selecting a new style or color triggers an automatic `click` on `#btn-run-search` if start/end fields are populated, ensuring real-time 3D feedback.
