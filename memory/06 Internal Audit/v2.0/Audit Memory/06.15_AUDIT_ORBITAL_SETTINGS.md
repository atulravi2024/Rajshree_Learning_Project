## 🏗️ SYSTEM ARCHITECTURE

The **Orbital Settings** replace the legacy static sidebar with a dynamic, immersive experience using 5 orbiting satellites. Each satellite represents a specific configuration domain and is linked to the shared project state (`window.CONFIG`).

### 1. Final Satellite Modules (v2.2 Standard)
- **System Ops**: Data resonance frequency, auto-sync heartbeat, and mission timer.
- **Visual Protocol**: NASA Texture resolution, Cloud opacity, Atmosphere Rayleigh scattering, and CRT scanline density.
- **User Gateway**: Auditor codename, permission tier, and session persistence toggle.
- **Neural Audio**: HD Interaction bleeps, environmental soundscapes, and AI voice guidance.
- **Security Guardrails**: Red-alert triggers, data masking, and hardware lockdown protocols.

### 2. Orbital Physics & Geometry (Deep Dive)
- **Torus Rings**: 3 distinct segments with `THREE.AdditiveBlending`:
  - **Ring 1**: Radius 118, Color `0x00f0ff` (Cyan), Speed `0.002`.
  - **Ring 2**: Radius 130, Color `0xfacc15` (Amber), Speed `-0.0015`.
  - **Ring 3**: Radius 143, Color `0x00f0ff` (Cyan), Speed `0.001`.
- **Label Projection**: 
  - Uses `sat.mesh.getWorldPosition(worldPos)` then `worldPos.project(camera)`.
  - Maps normalized device coordinates (-1 to +1) to pixel-space based on `renderer.domElement.getBoundingClientRect()`.
  - Updates `left`/`top` CSS styles for `.sat-label` elements on every `requestAnimationFrame`.

### 2. Orbital Ring Sync

### Step 1: Activation Logic
1. Triggered via the **Navbar Settings Gear icon**.
2. Globe scales to **0.65x** to make room for orbits.
3. `orbitalGroup.visible = true` and `.sat-label` elements are injected/revealed.

### Step 2: Interactive Panels
1. **Module Selection**: Clicking a satellite label triggers `openOrbitalPanel(id)`.
2. **Panel UI**: Glassmorphic holographic panels containing sliders (`input[type=range]`) and switches.

### Step 3: Global State Sync
1. **Change Detection**: Sliders use `input` listeners to call `applySettingEffect(id, val)`.
2. **Persistence**: Changes are synchronized across all modules (Dashboard, Map, Terminal) via the global `window.CONFIG` object.
3. **Immediate Feedback**: visual parameters (e.g., `--glow-opacity`) are updated in real-time on `document.documentElement`.

## 🤖 AI OVERSIGHT INSTRUCTIONS
1. **Sync Integrity**: When adding a new setting, ensure it exists in the default `CONFIG` object inside `shared/settings.js`.
2. **Label Physics**: Labels are 2D elements projected from 3D space. If labels "jump" or flicker, check the `renderer.domElement` bounding rect in the animation loop.
3. **Performance**: Only Lucide icons required for the active panel should be re-initialized using `lucide.createIcons({ scope: panel })`.
