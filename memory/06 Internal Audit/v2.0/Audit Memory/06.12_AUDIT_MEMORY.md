## 🏛️ INTERNAL HUB ARCHITECTURE (Deep Dive)

### 1. Technical Core (`window.CONFIG`)
- **State Management**: Centralized `CONFIG` object acts as the primary source of truth.
- **Sync Protocol**: Uses `postMessage(message, '*')` for cross-iframe synchronization between Settings and Map/Dashboard.
- **State Flags**: 
  - `_isDeepScanning`: Boolean gate for scan-in-progress.
  - `_isLockedDown`: Global system override status.
  - `_satSpeedMultiplier`: Float (0.0 - 5.0) for orbital velocity control.

### 2. Neural Console (Front Face)
- **Logic**: Dynamic `lucide.createIcons()` hydration on every DOM injection.
- **Visuals**: CSS filters (`backdrop-filter`, `hue-rotate`, `sepia`) for high-fidelity HUD states.
- **Resource Link**: Direct mapping to `06.13_AUDIT_DASHBOARD_DOMAINS.md`.


### 3. Audit Findings (Back Face)
