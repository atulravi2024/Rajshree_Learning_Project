## 🏗️ VISUAL RENDERERS (Canvas 2D Logic)

The holographic sidebars utilize multiple HTML5 Canvas elements for peak performance rendering of dynamic data visualizations.

### 1. Resource Donut Chart (`renderDonutChart`)
- **Logic**: Manual arc drawing on a `90x90` canvas.
- **Inner Hole**: Drawn with `InnerR = OuterR * 0.55` and filled with `rgba(4, 14, 22, 0.95)` to create a persistent donut face.
- **Sync**: Auto-populates a side legend with labels (CREDIT, DATA, SYNC, THREAT) and their corresponding colors.

### 2. Sector Flow Wave (`renderSectorFlow`)
- **Animation**: Continuous sine-wave loop using `requestAnimationFrame`.
- **Mathematical Formula**: `h / 2 + Math.sin(i / 20 + Date.now() / 500) * 15`.
- **Properties**: Cyan stroke (#00f0ff), `lineWidth: 2`, 10px crawl interval.

### 3. Signal Interference Noise (`renderSignalInterference`)
- **Animation**: High-frequency strobe effect (100ms interval).
- **Mechanism**: Randomly scatters `1x1` yellow pixels (#facc15) across a semi-transparent black background.

## ⚡ PHYSICAL SIMULATORS (Jitter Engine)

To prevent "static" UI, various widgets use randomized jitter logic to simulate real-time sensor fluctuation.

### 1. Hardware Thermal Jitter (`renderHardwareGauges`)
- **Logic**: Base temperature + Gaussian-style variance.
- **Math**: `Math.min(100, Math.max(10, m.base + Math.floor(Math.random() * m.variance * 2) - m.variance))`.
- **States**: `cool` (<55°C), `warm` (55-79°C), `hot` (>80°C - triggers Red CSS class).

### 2. Latency Matrix Jitter (`renderLatencyMatrix`)
- **Interval**: 2500ms cycle.
- **Classification**: 
    - `low-latency`: < 10ms.
    - `high-latency`: > 30ms (triggers Amber CSS class).
