## ⚙️ LOCKDOWN SEQUENCE (`triggerLockdown`)

The Lockdown Protocol is a global state override designed to freeze the interface and secure the system in response to a critical threat.

### Step 1: Interface Suppression
- **CSS Trigger**: Toggles `lockdown-active` and `stealth-mode-active` classes on `document.body`.
- **Icon Mutation**: Switches Lucide icon from `shield-off` to `shield` on `#btn-lockdown`.
- **Input Locking**: CSS `pointer-events: none` is applied to all primary navigation and input fields.
- **Orbital Freeze**: Sets `window.controls.enabled = false` to disable user manipulation of the 3D scene.

### Step 2: Session Sanitization (`purgeActiveSessionData`)
- **Action**: Immediately clears all persistent search inputs (`From`, `Via`, `To`).
- **Path Cleanup**: Calls `window.drawQuantumPath([])` to wipe active navigation traces from the globe.
- **Cache Flush**: Consoles confirm "[Lockdown] Session data purged."

### Step 3: Global Quarantine (`performGlobalQuarantine`)
- **Node Resolution**: Iterates through `window._mapGlobe.nodes`.
- **Scrubbing**: Any node with `userData.status === 'critical'` is forcibly reset to `resolved`.
- **Color Sync**: Materials are set to hex `0x22c55e` (Green) and scale reset to `(1, 1, 1)`.

### Step 4: Live Console Feed (`startLockdownLiveFeed`)
- **Interval**: 1800ms periodic update.
- **Message Stack**: Iterates through an 9-entry array of secure status messages (e.g., "NEURAL RADIANCE FIELDS MONITORING").
- **Visuals**: Uses `.prepend()` to push the latest logs to `#lockdown-live-logs` with color-coded classes (`info`, `warn`, `crit`).

## 🆘 MANUAL OVERRIDE PROTOCOL

A deliberate fail-safe allowing the Lead Auditor to bypass the lockdown through an intentional hold-pattern.

### Logic Flow (`startOverrideHold`)
1. **Hold Duration**: 45ms tick interval.
2. **Progress Calculation**: `holdProgress += 1.5` per tick.
3. **Completion Threshold**: 100% (~3000ms / 3.0 Seconds).
4. **Interactive Labels**:
    - `0% - 33%`: "CRACKING ENCRYPTION SEALS..."
    - `34% - 66%`: "BYPASSING NEURAL GATEWAYS..."
    - `67% - 99%`: "RESTORING SYSTEM KERNEL..."
5. **Auto-Trigger**: Once `holdProgress >= 100`, calls `completeOverride()` and toggles `triggerLockdown()` off.
