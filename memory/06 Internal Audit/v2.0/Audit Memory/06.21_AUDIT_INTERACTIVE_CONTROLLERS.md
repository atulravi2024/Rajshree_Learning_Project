## 📷 CAMERA ZOOM DYNAMICS (`globe-zoom-slider`)

The camera zoom is governed by a mapping function that translates a 0-100 UI slider value into a 50-500 unit World Space distance.

### Transformation Math
- **Slider to Camera Z (`mapSliderToZ`)**: 
    - Translates 0 -> 500 (Far) and 100 -> 50 (Near).
    - `window._mapTargetCameraZ = mapSliderToZ(parseFloat(e.target.value));`
- **Camera Z to Slider (`mapZToSlider`)**: 
    - Reversed logic for programmatic UI updates (e.g., during initialization or UI Minimize).

### UI Minimize Event (`btn-ui-toggle`)
- **Action**: Toggles `ui-minimized` body class.
- **Auto-Zoom**: 
    - **Minimized**: Set `camera.z` to `150` for focused detail.
    - **Normal**: Set `camera.z` to `250` for global context.
- **Sync**: Dispatches `window.dispatchEvent(new Event('resize'))` after `520ms` (Matching the CSS transition duration).

## 🛰️ SATELLITE ROTATION MULTIPLIER (`sat-speed-slider`)

Interactive control over the orbital velocity of the 5-satellite constellation.

### Mathematical Sync
- **Speed Variable**: `window._satSpeedMultiplier = parseFloat(e.target.value);`
- **Rotate Speed Logic**: 
    - `window.controls.autoRotate = val > 0;`
    - `window.controls.autoRotateSpeed = 0.5 * val;`

## 🎨 VISUAL SYNKRONIZATION (`btn-visual-boost`)

A state-driven toggle that modifies multiple Three.js shader parameters simultaneously.

### Boost State Logic
- **Flag**: `window._isVisualBoostActive`.
- **Toggle**: `btn-visual-boost.classList.toggle('active')`.
- **Effect Routing**: Calls `window.applyVisualBoostState()` to re-calculate emissive and ambient intensities (Specs in **06.19**).
