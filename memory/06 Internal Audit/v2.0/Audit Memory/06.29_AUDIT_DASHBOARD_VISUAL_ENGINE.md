## 🌈 RGB SYNCHRONIZATION (`updateVisualSettings`)

The Frontier Hub uses a complex CSS-in-JS synchronization bridge to ensure accent colors remain consistent across shadow-DOM boundaries and legacy pseudo-elements.

### 1. Root Variable Injection
- **Mechanism**: Extracts `--accent-rgb` from the body compute-style and normalizes it to a `rgb()` string.
- **Propagation**: 
    - Forced update of `:root` variables: `--theme-accent`, `--cyan`, `--text-cyan`.
    - Calculation of `--cyan-dim` using current RGB triplets with `0.6` alpha.

### 2. Dynamic Style Tag Injection
- **Target**: `#frontier-dynamic-theme` (head-mounted).
- **Logic**: Injects raw CSS rules for `.nav-link.active::after` and `::-webkit-scrollbar-thumb` using the active RGB values and intensity multipliers:
    - **Soft Glow**: `0.2 * (glowIntensity / 100)`.
    - **Mid Glow**: `0.4 * (glowIntensity / 100)`.

## 📉 KPI JITTER ENGINE (`initDataRefresh`)

To simulate live data throughput, the dashboard features a non-destructive jitter simulation.

- **Trigger**: Controlled by `window.CONFIG.autoSync` and `window.CONFIG.refreshRate`.
- **Logic**:
    1. Selects all `.domain-kpi` elements.
    2. Stores the original value in `data-base` to prevent value drift over time.
    3. Formula: `newVal = Math.max(0, (base + (Math.random() - 0.5) * 4)).toFixed(0)`.
- **Persistence**: Jitter is additive and recalculated on every interval (default 2.5s) ensuring the "Pulse" of the project remains visible.
