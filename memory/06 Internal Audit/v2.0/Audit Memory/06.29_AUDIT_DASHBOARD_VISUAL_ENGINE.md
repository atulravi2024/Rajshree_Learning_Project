## 🌈 RGB SYNCHRONIZATION (`updateVisualSettings`)

The Frontier Hub uses a complex CSS-in-JS synchronization bridge to ensure accent colors remain consistent across shadow-DOM boundaries and legacy pseudo-elements.

### 1. Root Variable Injection
- **Mechanism**: Extracts `--accent-rgb` from the body compute-style and normalizes it to a `rgb()` string.
- **Propagation**: 
    - Forced update of `:root` variables: `--theme-accent`, `--cyan`, `--text-cyan`.
    - **Logic**: `--cyan-dim` is set to `rgba(${rgb}, 0.6)` for secondary UI elements.
- **Performance**: Wrapped in `requestAnimationFrame` to ensure zero-latency application during theme shifts.

### 2. Dynamic Style Tag Injection
- **Target**: `#frontier-dynamic-theme` (Verified ID).
- **Logic**: Injects raw CSS rules for `.nav-link.active::after` and `::-webkit-scrollbar-thumb`.
- **Glow Architecture**: 
    - **glowSoft**: `0.2 * (glowIntensity / 100)` (20% Alpha).
    - **glowMid**: `0.4 * (glowIntensity / 100)` (40% Alpha).

## 📉 KPI JITTER ENGINE (`initDataRefresh`)

To simulate live data throughput, the dashboard features a non-destructive jitter simulation.

- **Trigger**: `initDataRefresh()` governed by `refreshRate` (Default: 2.5s).
- **Logic**:
    1. Targets all `.domain-kpi` and `.footer-val` elements.
    2. Uses `el.dataset.base` for absolute persistence to prevent cumulative drift.
    3. Formula: `newVal = Math.max(0, (base + (Math.random() - 0.5) * 4)).toFixed(0)`.
- **Fidelity**: Simulates a live diagnostic "Pulse" without modifying the underlying `DOMAIN_DATA` state.

## 🎨 SENTINEL RENDERER (Charts & Waves)

The visual engine handles two primary high-fidelity data visualizations via `renderCardChart`.

### 1. Chart.js Integration
- **Types**: `line` (Area-style), `bar` (Standard blocks).
- **Styling**: "Sparkling" linear gradients (top: `glowMid`, bottom: transparent).
- **Configuration**:
    - **Easing**: `easeOutQuart` (800ms duration).
    - **Points**: `pointRadius: 0` (performance optimization), `hoverRadius: 4`.
    - **Shadows**: Custom `shadowBlur: 10` for area charts to enhance depth.

### 2. Neural Waveforms (`cardType: 'waveform'`)
- **Mechanism**: CSS-animated `div.wave-bar` elements (used in Sector 09).
- **Pulse Logic**: Randomized `animation-duration` (1.2s - 1.8s) and `animation-delay` (up to -2s) create an asynchronous "Neural Ripple" effect across 60+ bars.
