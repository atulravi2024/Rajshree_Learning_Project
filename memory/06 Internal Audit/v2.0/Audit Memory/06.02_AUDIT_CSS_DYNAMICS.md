## 🎨 STYLING WORKFLOW (Step-by-Step)

### Step 1: Token Engine (`:root`)
- **Semantic Multipliers**: Glow values are functionally dynamic: 
  - `rgba(var(--accent-rgb), calc(0.2 * var(--neural-glow-opacity)))`.
- **Glow Hierarchy (0.85 Default)**: 
  - `subtle`: 10% base opacity.
  - `soft`: 20% base (Standard UI halo).
  - `mid`: 40% base (Active/Hover card focus).
  - `max`: 95% base (Critical alert flare).
- **Persistence**: Variables are synchronized between `dashboard.css` and `visual_sync.js`.

### Step 2: Atmospheric Layering (Source: `background.css`)
1. **Structural Grid**: `55px` repetitive pattern using `linear-gradient` center offsets.
2. **Holographic Glare**: Multi-stop radial gradient for top-down depth.
3. **Scanline Layer**: `4px` interval repeating gradient with `0.4` opacity management.
4. **Noise Layer**: High-frequency grain texture for "analog" fidelity simulation.

### Step 3: Neural Console Glassmorphism
- **Filters**: `backdrop-filter: blur(var(--glass-blur))` (Default: 10px).
- **Borders**: `1px solid var(--border-glow)` (reacts to color theme shift).
- **Shadows**: `-4px 0 30px rgba(0, 0, 0, 0.5)` for multi-z-layer separation.

### Step 4: The 1.3x Launch Protocol
1. **Interactive Scale**: `.btn-launch:hover` uses `transform: scale(1.3) translateX(-40px)`.
2. **Clipping Prevention**: Parent `.console-body` uses `overflow: visible` during hover events via JS delegation.
3. **Motion Curves**: Transitions use `cubic-bezier(0.175, 0.885, 0.32, 1.25)` for fluid HUD feedback.
4. **Performance**: `will-change: transform` applied to prevent GPU stutter.

### Step 5: Registry & Themes
1. **Theme Registry**:
   - `.theme-frontier` (Cyan), `.theme-obsidian` (Purple), `.theme-nordic` (Frost).
2. **Ghost Theme Fix**: 
   - **Contradiction**: Light themes require dark text on hover.
   - **Solution**: `.theme-ghost .domain-card:hover` overrides all labels to `#1a2a3a` to maintain contrast.
3. **Status Animators**:
   - `sentinel-aura` (Stable), `sentinel-glitch` (Active), `sentinel-spin` (Warning), `sentinel-alarm` (Threat).
