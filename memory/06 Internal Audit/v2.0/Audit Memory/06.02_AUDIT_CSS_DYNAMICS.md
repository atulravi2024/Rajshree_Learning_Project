## 🎨 STYLING WORKFLOW (Step-by-Step)

### Step 2: Cinematic Background Setup
1. **Void Gradient**: Apply a 3-layer radial/linear gradient blend for the deep space effect.
2. **Diagnostic Grid**: Set `background-size: 55px 55px` to match mockup density.
3. **Scanlines**: Apply a repeating linear gradient (0deg) with 4px intervals for the CRT effect.

### Step 3: Neural Console Glassmorphism
- **Step**: Apply `backdrop-filter: blur(12px)` and `border-left: 1px solid var(--border-glow)`.
- **Shadows**: Use `box-shadow: -4px 0 30px rgba(0, 0, 0, 0.5)` to create separation from the grid.

### Step 4: The 1.3x Launch Animation "Magic"
1. **Target**: `.btn-launch` inside the sidebar.
2. **Problem**: Default `overflow: auto` on `.console-body` clips the button when it scales.
3. **Step-by-Step Fix**:
   - Set `transition: overflow 0s` on parent.
   - Use `:has(.btn-launch:hover)` to toggle `overflow: visible` dynamically.
   - Ensure `min-height: 0` is set to prevent layout recalculation flicker.
