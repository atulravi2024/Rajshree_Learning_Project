# 🛑 EXCEL & CSV FILE GUARDRAIL (v1.0)

> **PRIMARY RESTRICTION:** The AI is strictly prohibited from interacting with any Excel (`.xlsx`, `.xls`) or CSV (`.csv`) files unless explicitly instructed by the user in the current conversation.

## 🚫 Restricted Actions
- **NO CHECKUPS:** Do not open, read, or verify the contents of Excel/CSV files autonomously.
- **NO CREATION:** Do not create new Excel/CSV files unless specifically asked.
- **NO UPDATION:** Do not modify, append, or update any Excel/CSV files (including the Master Database) without an explicit command from the user.

## ✅ Exceptions
- Only perform operations on these files if the user explicitly mentions the file name or gives a direct instruction to "update the excel file", "read the csv", etc.
- If a task seems to require an Excel update to be complete (as per the "Master Source" rule in `00.01_STRICT_BOUNDARIES.md`), the AI **MUST ASK FOR PERMISSION** before touching the Excel file.
