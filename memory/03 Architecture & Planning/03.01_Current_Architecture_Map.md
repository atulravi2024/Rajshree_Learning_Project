# Current Project Architecture (v17.2)

This document provides a pixel-perfect map of the Rajshree Learning Project's physical directory structure following the v17.0 modernization.

## 🏗️ CLEAN ARCHITECTURE
The project has been reorganized into specialized domains to eliminate fragmentation and prepare for production deployment.

---

## Technical Improvements
1. **Frontend Isolation**: All web-facing code is encapsulated within `frontend/`.
2. **Asset Pipeline**: Backend scripts are separated from the live web server assets.
3. **Cognitive Integrity**: The `memory/` folder is now the absolute source of truth for AI agents.
4. **Build Readiness**: The structure allows for easy bundling of `frontend/public` for static hosting.
