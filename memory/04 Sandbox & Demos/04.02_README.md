# 🗃️ `memory/04 Sandbox & Demos/` Namespace

> **WARNING**: The files within this `demo/` namespace track assets, methodologies, and prototypes that are expressly isolated from the main live application. 

### Purpose
When we build out separate prototype products (like the `Audio Demo/` TTS evaluation system), those products undergo rapid iteration and clutter-generation that corrupts the main `01.04_PROJECT_ASSETS.md` inventory. 

To maintain the architectural integrity of the live application (`Rajshree Learning Project`), ALL documentation regarding these side-projects must live permanently in this `demo/` memory enclave.

### Primary Tracker
*   **`DEMO_ASSETS.md`**: Maps the entire `Audio Demo/` asset landscape.
